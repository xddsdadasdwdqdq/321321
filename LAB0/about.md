Git используется для системы контроля версий. Он позволяет отслеживать изменения в коде, возвращаться к предыдущим версиям, работать над проектом нескольким разработчикам одновременно и безопасно хранить историю изменений.
GitHub Classroom используется для организации учебных заданий. Он автоматически создает репозитории для студентов, позволяет преподавателю проверять работу, а студенту — отправлять задания через Git.
