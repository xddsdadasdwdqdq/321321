git clone — копирует удалённый репозиторий на локальный компьютер  
git status — показывает текущее состояние репозитория и изменённые файлы  
git add — добавляет файлы в индекс для последующего коммита  
git commit — сохраняет изменения в локальной истории репозитория  
git push — отправляет локальные коммиты в удалённый репозиторий
